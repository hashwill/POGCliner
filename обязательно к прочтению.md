# POGCliner

🧼 PitOfGoblin CLiner — Инструкция по установке и использованию

📦 Описание

PitOfGoblin CLiner — это лёгкая утилита для отчистки и оптимизации вашего компьютера.

💻 Требования
Windows 10/11
20-50 MB свободного места

⚙️ ИНСТРУКЦИЯ ДЛЯ УСТАНОВКИ POGCliner

Скачав проект с GitHub извлеките на рабочий стол!(при извлечении в другие папки происходит краш POGCliner)

Открой PowerShell

Вставте две команды:

pip install pyinstaller

pyinstaller --onedir --windowed --name "launcherPOG" --collect-all tkinter install.py

Подожди 30 сек - создастся папка dist/
Запусти: dist\launcherPOG\launcherPOG.exe от администратора

🧼 Использование

После запуска откроется интерфейс “POG Cliner”.

Доступные функции:

🔹 Scan System

Полный анализ системы:

Ищет давно не используемые файлы и удаляет их с вашего разрешения!
Проверка на сторонее ПО
В конце проверки выводит все подозрительные файлы

🔹 Clean System

Запускается процесс отчистки:

Удаляет ненужные файлы после разрешения пользователя!
Увеличивает производительность на 10-15% после отчистки


🔹 Goblin Mode

Активирует HARD Проверку по всей системе:


🧠 Пример использования
[CLiner] Scanning system...
Found: 4837 junk files (mostly goblin residue)

[CLiner] Cleaning...
Removing imaginary cache... DONE
Optimizing chaos levels... DONE

System status: 99% more aesthetic ✨
